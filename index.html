Git Collaboration

Our team is practicing essential Git workflows for effective collaboration. Here are some of the commands we're using in this lab:

$ git clone https://github.com/team/empowercrafts-ghana.git # Clone the repository
$ git branch feature-dashboard # Create a new branch
$ git checkout feature-dashboard # Switch to the new branch
$ git add . # Stage changes
$ git commit -m "Add user dashboard features" # Commit changes
$ git push origin feature-dashboard # Push to remote branch
$ git checkout main # Switch back to main branch
$ git pull origin main # Get latest changes
$ git merge feature-dashboard # Merge feature branch into main

About EmpowerCrafts Ghana

EmpowerCrafts Ghana is a project aimed at empowering local artisans by providing digital tools to showcase and sell their crafts globally. Our team is developing a platform that connects Ghana's talented craftspeople with international markets.

Through our Git collaboration exercise, we're building the foundation for this platform while practicing essential software engineering skills.
